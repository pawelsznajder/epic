\doxysubsection{Detailed Description}
Definition of enumeration values for writers available in Hep\+MC3. 

This class defines a set of enumeration values that are used to distinguish between writer availibles in Hep\+MC3 library. In addition, a declared object of this class is always associated to one type (see Writer\+Hep\+MC3\+Type\+::m\+\_\+type), so member functions can act on it.
