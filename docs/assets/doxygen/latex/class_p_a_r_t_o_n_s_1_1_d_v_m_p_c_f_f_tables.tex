\doxysubsection{Detailed Description}
Module for evaluation of CFF values from lookup tables. 

Module for evaluation of CFF values from lookup tables (xi, t, Q2). Available tables are generated with PARTONS.

User may change between available sets of tables with \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_v_m_p_c_f_f_tables_a17967a6153195328656693ddef1cac86}{DVMPCFFTables\+::set\+CFFSet\+File()}}, or DVMPCFFTables\+::configure() using \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_v_m_p_c_f_f_tables_aa82790e380e8c93912b3dd703492338b}{DVMPCFFTables\+::\+PARAMETER\+\_\+\+NAME\+\_\+\+SET\+\_\+\+FILE}} tag.

By default \char`\"{}data/\+DVMPCFFTables/pi0\+\_\+table\+\_\+\+GK.\+root\char`\"{} tables are probed.

The interpolation over lookup table is done with linterp library (\href{https://rncarpio.github.io/linterp}{\texttt{ https\+://rncarpio.\+github.\+io/linterp}}).
