\doxysubsection{Detailed Description}
Definition of enumeration values for axis types. 

This class defines a set of enumeration values that are used to distinguish between axis types. In addition, a declared object of this class is always associated to one type (see Axis\+Type\+::m\+\_\+type), so member functions can act on it.
