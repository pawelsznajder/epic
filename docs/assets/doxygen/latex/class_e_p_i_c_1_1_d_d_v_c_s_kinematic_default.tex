\doxysubsection{Detailed Description}
Kinematic module to determine four-\/momenta from DDVCS kinematics. 

This class defines kinematic module for DDVCS process. That is, four-\/momenta of particles and the signature of event are defined here from kinematics and kinematic conditions.
