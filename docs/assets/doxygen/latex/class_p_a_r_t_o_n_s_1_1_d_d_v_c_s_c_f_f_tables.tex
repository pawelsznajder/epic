\doxysubsection{Detailed Description}
Module for evaluation of CFF values from lookup tables. 

Module for evaluation of CFF values from lookup tables (xi, t, Q2, Q\textquotesingle{}2).

Since the tables are for xi, Q2, Q\textquotesingle{}2 it is explicitly assumed that mu\+F2 = mu\+R2 = Q2 + Q\textquotesingle{}2 and eta = xi $\ast$ ((Q2 -\/ Q\textquotesingle{}2) / (Q2 + Q\textquotesingle{}2))

User may change between available sets of tables with \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_d_v_c_s_c_f_f_tables_af4c72ace0b213d322383aae45052a053}{DDVCSCFFTables\+::set\+CFFSet\+File()}}, or DDVCSCFFTables\+::configure() using \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_d_v_c_s_c_f_f_tables_a7f8f297eb9f5de84af59ca7ffb043309}{DDVCSCFFTables\+::\+PARAMETER\+\_\+\+NAME\+\_\+\+SET\+\_\+\+FILE}} tag. E.\+g. \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_d_v_c_s_c_f_f_tables_af4c72ace0b213d322383aae45052a053}{DDVCSCFFTables\+::set\+CFFSet\+File}}(\char`\"{}data/\+DDVCSCFFTables/tables\+\_\+\+GK.\+root\char`\"{}) will probe tables stored in \char`\"{}data/\+DDVCSCFFTables/tables\+\_\+\+GK.\+root\char`\"{} file.

By default \char`\"{}data/\+DDVCSCFFTables/tables\+\_\+\+GK.\+root\char`\"{} tables are probed.

The interpolation over lookup table is done with linterp library (\href{https://rncarpio.github.io/linterp}{\texttt{ https\+://rncarpio.\+github.\+io/linterp}}).
