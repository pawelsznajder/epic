\doxysubsection{Detailed Description}
Definition of enumeration values for particle types. 

This class defines a set of enumeration values that are used to distinguish between particle types. In addition, a declared object of this class is always associated to one type (see Particle\+Type\+::m\+\_\+type), so member functions can act on it.
