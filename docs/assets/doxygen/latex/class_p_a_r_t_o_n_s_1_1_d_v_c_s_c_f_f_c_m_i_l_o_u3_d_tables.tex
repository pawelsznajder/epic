\doxysubsection{Detailed Description}
Module for evaluation of CFF values from lookup tables. 

Module for evaluation of CFF values from lookup tables (xB, t, Q2). Available tables are imported from MILOU MC generator (\href{https://eic.github.io/software/milou.html}{\texttt{ https\+://eic.\+github.\+io/software/milou.\+html}}) variation for 3D tables (\href{https://drf-gitlab.cea.fr/monte-carlo/milou}{\texttt{ https\+://drf-\/gitlab.\+cea.\+fr/monte-\/carlo/milou}}).

Since the tables are for xB and Q2 it is explicitly assumed that xi=x\+B/(2-\/xB) and mu\+F2 = mu\+R2 = Q2.

User may change between available sets of tables with \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_v_c_s_c_f_f_c_m_i_l_o_u3_d_tables_a8328212f9a0cb3e8a860ff45a76e4183}{DVCSCFFCMILOU3\+DTables\+::set\+CFFSet\+File()}}, or DVCSCFFCMILOU3\+DTables\+::configure() using \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_v_c_s_c_f_f_c_m_i_l_o_u3_d_tables_aac5f523fc76c2977f5fc5a32ea85d815}{DVCSCFFCMILOU3\+DTables\+::\+PARAMETER\+\_\+\+NAME\+\_\+\+SET\+\_\+\+FILE}} tag. E.\+g. \mbox{\hyperlink{class_p_a_r_t_o_n_s_1_1_d_v_c_s_c_f_f_c_m_i_l_o_u3_d_tables_a8328212f9a0cb3e8a860ff45a76e4183}{DVCSCFFCMILOU3\+DTables\+::set\+CFFSet\+File}}(\char`\"{}data/\+DVCSCFFCMILOU3\+DTables/tables\+\_\+\+KM20.\+root\char`\"{}) will probe tables stored in \char`\"{}data/\+DVCSCFFCMILOU3\+DTables/tables\+\_\+\+KM20.\+root\char`\"{} file.

By default \char`\"{}data/\+DVCSCFFCMILOU3\+DTables/tables\+\_\+\+GK.\+root\char`\"{} tables are probed.

The interpolation over lookup table is done with linterp library (\href{https://rncarpio.github.io/linterp}{\texttt{ https\+://rncarpio.\+github.\+io/linterp}}).
